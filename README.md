## 상세

프랜차이즈 가맹점주용 관리어플리케이션. 어플리케이션을 통해 원격개점, 폐점, 정산, 통계를 볼 수 있습니다.

## 환경

- Android Studio
- Java 11

## 적용 예정 기술

1. Room Database
2. Navigation Bar Activity
3. Chart Library (회의 후 적용 lib 선정)
4. Android BioAuthentication
